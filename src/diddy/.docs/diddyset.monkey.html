Class DiddySet<T> Extends Set<T> Implements IContainer<T>
Description:The DiddySet class extends the official Monkey Set class and implements Diddy's IContainer interface.
Details:As with the other Diddy container classes, it simplifies mixing and matching of container types by sharing
common method names.
Note that since Set is unordered, many of the IContainer methods are non-applicable or have undefined results.
