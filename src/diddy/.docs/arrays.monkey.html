Class Arrays<T>
Description:The Arrays class provides utility functions for manipulating Monkey arrays.
Details:For example, you could clone the contents of an array, or join multiple arrays into one.
Note: Any functions that copy values are a "shallow copy". This means that for arrays of objects, only the object *reference* is copied to the new array. It is still the same object.

Function Clone:T[](arr:T[])
Description:The Clone function will create a new array with the same dimensions as the passed array, and copies each value from the old array to the new array.
Returns:T
Details:Note that this is a shallow copy, so objects in the array will not be duplicated.
Only their references will be copied.
' returns a new array with the same contents [1,2,3]
Arrays<Int>.Clone([1,2,3])

Function Copy:Void(src:T[], srcPos:Int, dest:T[], destPos:Int, length:Int)
Description:The Copy function will copy a range of values between two arrays (or within the same array).
Returns:Void
Details:
' copies 2 values from index 1 in src to index 2 in dest. dest now contains [5,6,2,3]
Arrays<Int>.Copy([1,2,3,4], 1, [5,6,7,8], 2, 2)

Description:The CreateArray function will create an array of arrays (sometimes called a multi-dimension array),

Function Equals:Bool(arr1:T[], arr2:T[])
Description:The Equals function will perform an equality comparison on every element in the two passed arrays.
Returns:Bool
Details:It returns False if the arrays are of different length, or when it finds the first non-match.
Otherwise, it returns True.
' returns False, because arrays are different length
Arrays<Int>.Equals([1,2,3], [1,2])

' returns False, because arrays contain different values
Arrays<Int>.Equals([1,2,3], [4,5,6])

' returns True, because arrays contain the same values
Arrays<Int>.Equals([1,2,3], [1,2,3])

Function Fill:T[](arr:T[], val:T)
Description:The Fill function will set every element of the passed array to the specified value, then returns the array.
Returns:T
Details:Note that this does not create a new array. The return value will reference the same array that was passed in.
' fills the existing array and returns it ["d","d","d"] (it doesn't create a new array)
Arrays<String>.Fill(["a","b","c"], "d")
